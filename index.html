Robotics -
Self Driving
Car Project

I built a self-driving car, which works with UltraSonicSensors (USS).
UltraSonicWaves are sent which then bounces back and with an advanced formula, calculate the distance of the signal sent.
Based of that information the arduino board receives, the code decides with an if statement which of the wheels to give power or not.
